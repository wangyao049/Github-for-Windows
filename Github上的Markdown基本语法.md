# 二、字体   
* **加粗**   
要加粗的文字左右用两个\*号包起来   
* *斜体*   
要倾斜的文字左右用一个\*号包起来   
* ***斜体加粗***   
要倾斜加粗的文字左右用三个\*号包起来   
* ~~删除线~~      
要加删除线的文字左右分别用两个\~~号包起来
